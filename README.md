# Multiplayer Arena Shooter Prototype

This is an early-stage multiplayer arena shooter prototype made in **Godot** as a learning project to strengthen my networking skills. The project is primarily inspired by classic arena shooters such as *Quake*, *Unreal Tournament*, and *Halo*.

The goal is to create a flexible, low-level multiplayer framework that supports both LAN and internet play.  

## Planned Features
- Server browser support for LAN and online games  
- Projectile and hitscan weapons  
- Smooth prediction of player movement  
- Deathmatch and team deathmatch  

## Aspirational Features
- Bot support  
- Additional game modes  
- Prediction of client-fired projectiles  
- Lag compensation  

## Long-term (unlikely) Features
- Physics-based vehicles
- Dedicated server support  

> **Note:** This project is in early development and is not yet playable.
